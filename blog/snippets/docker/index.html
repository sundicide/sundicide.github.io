ps format

영구 적용

vi ~/.docker/config.json
{
  "psFormat": "ID\t{{.ID}}\nNAME\t{{.Names}}\nIMAGE\t{{.Image}}\nPORTS\t{{.Ports}}\nCOMMAND\t{{.Command}}\nCREATED\t{{.CreatedAt}}\nSTATUS\t{{.Status}}\n"
}

일시 적용

export FORMAT="..."

docker ps --format="$FORMAT"
